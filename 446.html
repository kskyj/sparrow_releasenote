분석 속도 개선

- xml generating 속도 개선

xml generating 시 오래걸리는 부분을 개선하였습니다.


- Engine defect id 처리 부분 변경

Engine defect id 처리하는부분을 수정하여 분석속도를 개선하였습니다.


- source / xml 파일 동작 개선

로컬 일경우 sparrow 폴더 안의 source, xml 파일들을 복사하지 않고 move로 하게 변경하여 속도를 개선하였습니다.
