Eclipse Plugin  개선

 

 

- 파일수집/NEST업로드 프로그레스 세분화

파일수집, NEST 업로드 단계에서 프로그레스를 세분화 하였습니다.

 

- 라이선스 메시지 수정

라이선스 초과시 제공하는 메시지를 수정하였습니다.

 

 

 

기타 개선

 

 

- rmi 에러시 로그 수집하도록 개선

rmi를 통한 분석시 에러가 발생할 경우 로그를 수집하게 변경하였습니다.

 sparrow4/logs 폴더에 파일 생성

 upload-failed-whistle.log : 에러 발생한 시점의 sparrow-whistle.log를 백업해둠 20개까지

 upload-failed-engine-client.log : 엔진 클라이언트 upload 메소드 진행 상황을 상세히 기록함. 에러가 아니면 파일 기록안함

 

 

- Semantic 분석시 summary 파일 처리 개선

semantic engine이 전분석단계에서 summary 파일을 생성시 디렉토리 세분화 해서 생성하였습니다.
