엔진 개선

 

 

- OVERRUN.STATIC.MEMSET 정확도 향상

아래와 같은 사항에서 정확도 향상

Case 1) 구조체 변수의 주소가 dest일 때

struct ss {

char a[10];

….

}

 

void foo() {

struct ss s;

bzero(&s, sizeof(struct ss)); // 여기서 구조체 변수의 주소를 취하면, 항상 dest size가 11이라고 나와서 false alarm 이 발생(src size는 정상)

}

 

Case2) 2차원 배열

Void foo() {

Char tmp[10][16];

bzero(tmp, sizeof(tmp)); // 이 경우 dest의 크가가 90으로 나와서 false alarm 발생(src size는 정상)

}

 

 

- Proframe 을 위한 Extend library 수정

Extend library를  수정하여 Proframe 에서 분석 정확도를 개선하였습니다.
