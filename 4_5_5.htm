엔진 개선

 

 

 

- C/C++ 신규 체커 추가

아래와 같은 체커가 추가되었습니다.

체커분류

체커명

위험도

설명

Concurrency

TRYLOCK_FAIL

Level 1

이미 lock된 리소스를 다시 pthread_mutex_trylock(pthread_mutex_t *mutex); 함수로 호출하는 경우

API Usage

NEGATIVE_FILE_DESCRIPTOR

Level 3

지정된 함수들의 특정인자가 음수일때 알람을 냄. 지정함수들은 CodeSonar 3.7 doc의 Negative File Descriptor에 나와 있음

Misuse

EXCESSIVE_STACK_DEPTH

Level 1

스택에 지정된 크기 이상의 버퍼를 할당할 수 없음

API Usage

NONEFFECTIVE_FUNCTION_CALL

Level 4

효과(memory effect)가 없는 함수 호출

 

 

- AIX 프리셋 통합

aix non standard 프리셋을 제거하고 AIX 에 통합하였습니다.

 

 

- BAD_INTERGER_EXPRESSION 기본 옵션 변경

16진수 (0x111)등에 대하여 검출하지 않게 변경
