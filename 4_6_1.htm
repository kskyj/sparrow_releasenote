NEST  개선

 

 

- 체커그룹 Excel 내보내기

체커그룹에서 현재 필터된 항목된 체커목록을 Excel 파일로 내보내는 기능이 추가되었습니다.

 

- 결함목록 Excel 내보내기 담당자 지정

특정 소스경로 매핑파일을 설정하면, 결함목록에서 Excel 내보내기시 매핑파일과 비교하여 담당자가 자동으로 지정됩니다.

이때 담당자는 NEST와 상관없이 매핑 파일 기준으로 동작합니다.

 

 

 

Whistle 개선

 

 

- Proc 분석 .c 업로드 하지 않도록 변경

Proc 분석시 .pc 파일이 있을경우 동일한 .c 파일은 업로드 하지 않도록 변경하였습니다.

 

- Java 분석 프로그레스 세분화

JSPC 분석 과정에 프로그레스를 세분화 하였습니다.

 

 

 

 

기타 개선
