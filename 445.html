엔진 개선

- OVERRUN.STATIC.MEMSET 정확도 개선

아래의 코드의 붉은색 부분에서 오류 내지 않음

typedef struct _A {

  ...

} A;


A a;

memset(&a, 0x00, sizeof(A));



- UNSUED_VALUE 정확도 개선

아래의 코드의 붉은색 부분에서 오류 내지 않음

int foo()

{

...

return 0;

}


int main()

{

...

foo();

return 0;

}



- FORMAT_STRING 정확도 개선

아래의 코드의 붉은색 부분에서 오류 내지 않음

int d;

scanf("%d",&d);

printf("%d",d);



- C++ 분석 개선

C++ 변환시 statements나 expression에서 알람이 나오는 경우 제거





기타 개선
