SPARROW Release ››

Released date 30 Nov. 2011


-- Component

- NEST C/C++/JAVA/JSP

- OPUS1 for C/C++

- WHSITE for C/C++

- WHISTLE GUI for C/C++

- WHSITE for JAVA (Beta)


-- Compatibility

- Eclipse 2.0

- Hudson Plugin 1.0 for C/C++

- Hudson Plugin 1.0 for JAVA (Beta)


--Requirement

*NEST

- JDK 1.5 or above


*WHISTLE

- JRE 1.5 or above


--Features

Added defect navigator

Added duplicated defect detection

Added detail analysis information page

Added defect description

Added custom setting for easy setting

Added project import

Added user permission

Support all encoding in report

Sharing classification all or each project

Support Visual studio solution(vs6~2010)

Upgrade Defect list UI for Convenience

Fixed many parsing error

Improved parsing error condition

Many bug fixed
