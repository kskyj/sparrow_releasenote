신규 페이지 추가

 

 

- 시스템 정보 표시 및 남은 용량 경고

NEST 가 설치된 하드웨어 정보를 표시하며, 남은 공간이 1GB 일경우, NEST 상단에 경고 표시를 합니다.

 

 

 

NEST 화면 개선

 

 

- Project / Scan Page 에 감춘 결함 제외하는 옵션 추가

숨겨진 결함 제외 체크박스를 추가하여 결함상태가 False , Intentional 로 표시한것들을 제외합니다.

체크하지 않은 경우 False, Intentional 도 개수에 포함됩니다.

 

 

- split 창 close 버튼 추가

split 로 이동한것이 종료 가능하도록 x 버튼을 추가하였습니다.

 

 

 

WhistleManger 개선

 

 

- 체커그룹 변경하는 기능 추가

휘슬매니저에서 NEST에 등록된 체커그룹을 변경 할 수 있습니다.

 

 

- Preset 이름 개선

Preset 이름을 실제 개발환경과 동일하게 변경하였습니다.

 

- 마법사에서 일반 모드 제거 및 Automation 추가

프로젝트 마법사에서 일반 모드를 제거 하였으며, Preset에 Automation 을 추가하였습니다.

 

 

 

 

Eclipse Plugin 개선

 

 

- 동일한 오류유형 감추는 옵션 추가

아래에서 알람의 근원은 getString() 메소드의 return null 구문입니다.

 

public class SimilarAlarmTest {

 public String getString() {

         return null;

 }

 

 public void test1() {

         String str = getString();

         

         str.toUpperCase();

 }

 

 public void test2() {

         String str = getString();

         

         str.toLowerCase();

 }

}

 

-show_follower 옵션이 없을 시에는 str.toUpperCase() 에서만 알람을 검출하고,

-show_follower 옵션이 있을 경우에는 str.toUpperCase(), str.toLowerCase() 에서 알람을 검출합니다

 

 

- system id 얻어오는 기능

시스템 아이디 얻어오는 기능을 추가

 

 

 

 

신규 개발 환경 지원

 

 

- MacOS 32bit Java 지원

MacOS 환경에서 32bit java를 사용할 경우 동작되도록 변경하였습니다.

 

- PCW IDE[CCSC] 일부 지원

CCSC PCW IDE에 관해서 일부 프로젝트를 지원합니다.

 

- 특정 Cygwin ucc 컴파일러 환경 추가 지원

특정 cygwin 에서 ucc컴파일러 환경을 추가 지원 합니다.

 

 

 

라이선스 / 권한 개선

 

 

- RMI 라이선스 체크 추가

NEST RMI, Engine RMI 실행시 라이선스 체크를 추가합니다. 즉 엔진추가시 라이선스를 발급 해야 합니다.
